- 홀수면 수, 짝수면 박이 나오게 하기

````javascript
function solution(n){
    var answer = "";
    for ( var i = 0 ; i < n ; i ++ ){
        if ( i % 2 == 0 ) {
            answer += "수";
        } else {
            answer += "박";
        }
    } return answer;
}
````
